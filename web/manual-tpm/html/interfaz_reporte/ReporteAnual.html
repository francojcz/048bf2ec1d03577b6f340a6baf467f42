Reporte anual

El reporte anual mediante la opción multireporte, le muestra información de tiempos, indicadores, pérdidas y muestras e inyecciones en gráficos de torta, barras y de dispersión, durante un año específico. Ver figuras 1, 2, 3 y 4.

Para configurar reporte mensual gráfico, seleccione el mes y el año y pulse el botón generar gráficos, si usted desea puede utilizar adicionalmente otros criterios tales como el equipo, método y/o analista para ver la información.
