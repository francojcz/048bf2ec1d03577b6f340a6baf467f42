Exportar datos de corridas analíticas

Para exportar los datos de corridas, seleccione el periodo entre fechas, y si es necesario el equipo y el analista, el sistema desplegara una lista en la parte inferior de la pantalla con los datos según la configuración dada por usted, si desea guardar los datos en formato de hoja de cálculo pulse el botón guardar en formato Excel el sistema descargará un archivo con los datos de la corrida que se mostraron en la lista.
