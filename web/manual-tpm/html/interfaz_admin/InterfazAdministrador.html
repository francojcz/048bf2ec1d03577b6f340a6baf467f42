MANUAL DE USUARIO
 TPM-QLabs  - INTERFAZ  ADMINISTRADOR



Una vez hayamos digitado correctamente la clave y la contraseña, el usuario administrador ingresara a una pantalla como se muestra en la figura 1.

Figura 1.Pantalla principal del administrador.

Básicamente la interfaz del  administrador está compuesta de los siguientes componentes:
(1) Cabecera donde se muestra el nombre de la empresa.
(2) Menú que muestra las opciones a las cuales puede acceder el administrador.
(3) Área de trabajo que muestra información según el menú que seleccione el usuario.
