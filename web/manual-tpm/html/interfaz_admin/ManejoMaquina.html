Manejo de equipos ó máquinas


MANUAL DE USUARIO
 TPM-QLabs  - MANEJO EQUIPOS



Cuando el usuario selecciona del menú la opción manejo de equipos, el sistema le desplegara una pantalla como la que se muestra en la figura 1. En esta pantalla el usuario podrá crear,  ver, actualizar, eliminar y restablecer los equipos.

Figura 1. Pantalla de manejo de equipos.
Agregar equipo

Para agregar un equipo pulse el botón Agregar, diligencie el formato que hay al lado derecho con la siguiente información y pulse el botón Guardar.
  • Código de inventario  :  Digite el código con el que se identificará el equipo en su inventario  (es un campo obligatorio).
  • Nombre del equipo  :  Digite el nombre del equipo  (es un campo obligatorio).
  • Estado  :  Seleccione el estado del equipo  (es un campo obligatorio).
  • Tiempo inyección  :  Digite el tiempo de inyección estándar del equipo  (no es un campo obligario, por defecto es 0). Este campo se tiene en cuenta para el calculo de algunos indicadores en los reportes, por tanto es necesario que sea diligenciado con información veridica y precisa.
  • Marca  :  Digite la marca del equipo  (es un campo obligatorio).
  • Modelo  :  Digite el modelo del equipo  (es un campo obligatorio).
  • Fecha de adquisición  :  Seleccione o digite en formato (AAAA-MM-DD) la fecha de adquisión del equipo  (es un campo obligatorio).
  • Computador  :  Seleccione el nombre del computador(nombre de certificado del computador) en el cual se registrarán los datos  (es un campo obligatorio).

    Actualizar

    Para actualizar un     equipo, seleccione de la lista el equipo que desea actualizar, actualice la información en el formulario del lado derecho y pulse el botón Actualizar. El sistema desplegara una pequeña ventana donde le preguntara porque razón desea actualizar el equipo, digite la razón y pulse el botón aceptar. Luego el sistema le mostrará un breve mensaje en el que informa del éxito del cambio.


    Eliminar

         Para eliminar un equipo, seleccione el equipo que desea eliminar de la lista y pulse el botón Eliminar. El sistema desplegara una pequeña ventana donde le preguntara si realmente desea eliminar la máquina y porque razón desea eliminarla, digite la razón y pulse el botón aceptar. Luego el sistema le mostrara un breve mensaje en el que informa del éxito de la eliminación.


    Restablecer

    Si usted ha eliminado un     equipo,  usted tiene la posibilidad de recuperar la información, seleccione la opción para ver los equipos eliminados (estrella blanca), seleccione de la lista el equipo que desea recuperar, y pulse el botón Restablecer, el sistema le preguntara porque razón usted desea restablecerlo, digite la razón y pulse el botón aceptar. Luego el sistema le mostrara un breve mensaje en el que informa si se ha restablecido con éxito.
