 TPM-QLabs  - INTERFAZ ANALISTA


Una vez hayamos digitado correctamente la clave y la contraseña, el usuario analista ingresara a una pantalla como se muestra en la figura 1.
La interfaz del analista está compuesta de los siguientes elementos.
  1. Cabecera  :  Nombre y logo de la empresa.
  2. Área de trabajo  :  Área de trabajo para ingresar los datos de las corridas analíticas.
  3. Ayudas  :  Ayudas conceptuales sobre los indicadores.
  4. Gráfica de tiempos  :  Gráfico
  5. Barra informativa y de utilidades  :  Muestra los datos del operario, la máquina seleccionada, el día,  y botones para salir, ingresar a ver los reportes entre otros.
  6. Barra de botones  :  Tiene botones para registrar nuevas corridas y manejar eventos.
