A.  INFORMACIÓN PARA EL SET DE ANALISIS

1. Tiempo entre Métodos

Es el tiempo medido entre el momento en que termina una corrida analítica, incluyendo el lavado del sistema, y el tiempo en que se inicia el alistamiento de la siguiente. Se toma como tiempo no programado (TNP).

El dato se ingresa en formato HH:MM (horas:minutos). El campo inmediatamente inferior es un cálculo utilizado para convertir el valor HH:MM a horas solamente.

Tiempo entre Metodos   (Hrs)

10:17 Tiempo entre Métodos (Ingrese en formato HH:MM)
10.283 Tiempo entre Métodos en formato horas

2. Cambio de Método

Es el tiempo que toma el alistamiento del equipo (lavado, purga, estabilización) antes de iniciar una corrida analítica. Este tiempo se toma como “Pérdida por cambio de Modelo y Ajustes”. También incluye el tiempo de lavado del sistema posterior al análisis, cuando aplica. Esta pérdida se toma como tiempo programado (TPP) ó tiempo estándar utilizado para lavado y alistamiento pre y post-corrida analítica.

En este campo también se ingresa el tiempo destinado a mantenimientos programados.

Este tiempo se convierte en Paro no Programado (TPNP) cuando implica reinstalaciones, lavados, estabilizaciones ó purgas adicionales previas a la corrida analítica y se toma como “Paro Menor”. Esta pérdida es calculada y se muestra en el campo inferior adyacente.

Cambio de Método     (min)

30.0 Tiempo de alistamiento del equipo (Cargado por defecto en minutos o para Ingresar dato en minutos)
13.0 Tiempo de alistamiento adicional (Resultado de cálculo en minutos)

3. Tiempo de Corrida

Tiempo de corrida estándar para análisis de solución test (System Suitability), estándares y muestras. Este tiempo es predefinido para cada metodología analítica que se implementa en el equipo y se carga en el campo por defecto al elegir el método. En estos campos también se puede ingresar un tiempo de corrida diferente al preestablecido según se requiera en el análisis pero algunos cálculos serán realizados con base en el tiempo estándar predefinido para comparar y determinar así la pérdida o ganancia en la eficiencia del método.

En el campo inferior adyacente se registra por defecto el mismo valor del campo superior, cuando es necesario el ingreso de inyecciones adicionales o reinyecciones.

5. Hora de Inicio de Corrida

Registro de la hora en que se da inicio a la corrida analítica en formato HH:MM. Para equipos automáticos, este dato también puede ser tomado del registro de tiempo que lleva el software que controla el equipo (Hora en que se inicia el primer análisis).

Hora Inicio de Corrida  

10:50 Hora de inicio de la corrida analítica (Ingresar dato en formato HH:MM)

6. Hora de Fin de Corrida

Registro de la hora en que se termina la corrida analítica en formato HH:MM. Para equipos automáticos, este dato también puede ser tomado del registro de tiempo que lleva el software que controla el equipo (Hora en que se termina el último análisis).

Hora Fin de Corrida
