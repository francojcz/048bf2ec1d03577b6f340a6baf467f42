MANUAL DE USUARIO
 TPM-QLabs  - MANEJO DE SÚPER USUARIOS



Cuando el usuario selecciona del menú la opción manejo de usuarios, el sistema le desplegara una pantalla como la que se muestra en la figura 1. En esta pantalla el usuario podrá crear, ver, actualizar, habilitar y deshabilitar los usuarios con rol superadministrador.

Figura 1. Pantalla de manejo de usuarios.

Agregar usuario

Para agregar un usuario pulse el botón Agregar, diligencie el formulario que está en el lado derecho llenando la siguiente información y pulse el botón Guardar.
  • Login de usuario  :  Ingrese el login de usuario  (es un campo obligatorio).
  • Password  :  Ingrese el pasword debe ser mínimo de 8 dígitos (es un campo obligatorio).
  • Repetir password  :  Repita el password (es un campo obligatorio).
  • Perfil  :  Seleccione un perfil de usuarios, los perfiles definen que puede hacer cada persona, (es un campo obligatorio)
    • Súper administrador: Es un perfil reservado para Quantar labs, dentro de las tareas principales que puede desempeñar este perfil están manejo de empresas, usuarios y certificados.
  • Habilitado  :  Seleccione si el usuario está habilitado o deshabilitado, tenga en cuenta que los usuarios deshabilitados no podrán ingresar al sistema (es un campo obligatorio).

Actualizar

Para actualizar un usuario, seleccione un usuario de la lista, actualice la información en el formulario del lado derecho y pulse el botón Actualizar. El sistema desplegara una pequeña ventana donde le preguntara porque razón desea actualizar el usuario, digite la razón y pulse el botón aceptar. Luego el sistema le mostrara un breve mensaje en el que informa del éxito del cambio.


Deshabilitar

Para deshabilitar un usuario, seleccione un usuario de la lista y en el formulario del lado derecho seleccione en el campo habilita la opción No, y pulse el botón actualizar, El sistema desplegara una pequeña ventana donde le preguntara porque razón desea actualizarlo, digite la razón y pulse el botón aceptar. Luego el sistema le mostrara un breve mensaje en el que informa del éxito de la actualización, recuerde que los usuarios deshabilitados no podrán ingresar al sistema.
